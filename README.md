[Demo Video](https://drive.google.com/file/d/1-c0e8CrspoOyp1P8Lgc4mF_NKE1WtimM/view?usp=sharing)

## Team Members
- Felipe Zavala

## How to Run
1. Clone repository: `git clone https://github.com/fezavala/Java-2D-Platformer-Project`
2. Open in IntelliJ
3. Run `GameController.java` in src/main/java/controller

## Features Implemented
- Feature 1: Player Character
- Feature 2: Physics System
- Feature 3: Collision Detection
- Feature 4: 5 Levels with solid platforms, coins, hazards, and a goal
- Feature 5: Game Loop
- Feature 6: HUD Display with collectible count and current time
- Feature 7: Restart level or play new level after beating a level
- Feature 8: Level data saves for most collectibles collected and best time
- Feature 9: A Python based level editor used to create the games levels

## Controls
- WASD / Arrow Keys: Move left and right
- Space: Jump

## Known Issues
- Issue 1: Game may visually stutter if system is not powerful
- Issue 2: Rendering inefficiencies
